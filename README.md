# Food Ordering Analytics Dashboard

## Project Overview

This project analyzes food ordering behavior using Tableau.

### Dashboard KPIs

- Total Orders
- Total Revenue
- Average Order Value
- Average Delivery Time

### Visualizations

- Orders by City
- Revenue by City
- Orders by Cuisine
- Revenue by Cuisine
- Orders by Meal Type
- Customer Age Distribution
- Repeat vs New Customers
- Orders by Weather

## Tools Used

- Tableau Public
- GitHub
